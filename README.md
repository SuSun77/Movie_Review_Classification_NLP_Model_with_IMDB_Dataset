# IMDB   
RNN model with preprocessing on the original IMDB dataset.   
Data can be downloaded from http://ai.stanford.edu/~amaas/data/sentiment/   
Folder structure:   
* data   
  * aclImdb   
    * test   
      * pos   
      * neg   
    * train   
      * pos   
      * neg   
* models   
  * saved model   
* test_NLP.py   
* train_NLP.py
